# 📊 Vendor Performance Analysis using Python and Power BI  

## 📌 Overview  
This project focuses on analyzing vendor performance using sales data to generate actionable business insights.  

It covers the complete analytics workflow including data ingestion, data cleaning, exploratory data analysis (EDA), automation of vendor-level summaries, and interactive dashboard development using Power BI.  

The project concludes with a professionally structured report summarizing key findings and performance metrics.

## 🚀 How to Run  

1. Clone the repository  
2. Open Jupyter Notebook and run the notebooks inside the `/notebooks` folder  
3. Execute `get_vendor_summary.py` to generate vendor-level summaries  
4. Open the `.pbix` file in Power BI to view the interactive dashboard  
5. Review the final insights in the `/reports` folder
